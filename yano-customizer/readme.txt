The easiest framework for Theme Developer in using WordPress Customizer.

== Description ==

What is Yano? "Yano" is a filipino word means easy, so Yano is a tool for WordPress Theme Developer in order to develop theme using WordPress Customizer API easily while writing clean and minimal code only.

Yano is developed to help WordPress Theme Developer to use WordPress Customizer API easily while writing minimal, maintainable code and also to avoid repetition of code.
